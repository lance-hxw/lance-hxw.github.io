即，可以用于检索数据的数据结构，比如用于数据库等场景

主要有
- b tree
	- mysql， pg，mongo
- hash table
	- redis hash等
- skiplist
	- redis zset
- R tree：多维数据
	- 地理信息等
- Trie tree: 处理字符串
	- 搜索引擎和全文索引
- 倒排索引
	- es
	- mysql fulltext
- bitmap 低基数字段
- lsm tree：写密集场景
	- rocksdb，leveldb
- AVL 严格平衡二叉树：内存
	- 要求左右子树高度差不超过1
- 红黑树 ：内存
	- 要求左右高度差不超过一倍
