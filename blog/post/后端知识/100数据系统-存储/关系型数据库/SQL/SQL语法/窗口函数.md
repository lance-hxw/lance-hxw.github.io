保留原始行并进行聚合计算

窗口函数可以访问与当前行相关的一组行， 并再不改变结果集行数的情况下进行计算
- group by 聚合后会变成若干组的数据
- 窗口函数是对每个组的row添加该组的新数据

基本语法是
函数() OVER (
	\[PARTITION BY col]
	\[ORDER BY col]
	\[ROWS/RANGE 窗口范围]
)

其中partition by 是将结果集按列分成若干窗口， 在每个窗口内单独计算

比如原来有abc三个列， 现在我想按b分组计算c的均值， 就可以
select a， b， c , avg(c) over (PARTITION BY b) as avg_c
from t

## 有哪些窗口函数
排名
- row_number， 给每行分配为一连续序号
- rank, 排名， 有并列
- DENSE_RANK, 排名， 并列且连续
- NTILE（n）， 将结果集分成n个组， 返回组号

偏移
- LAG获取当前行之前某行
- LEAD 之后某行
- FIRST_VALUE，窗口中第一行
- LAST_VALUE， 窗口最后一行
- NTH_VALUE， 窗口第n行

聚合
- sum， avg，count， min，max
分布函数
- PERCEDNT_RANK()百分位排名

## 如何划分窗口
partition by： 按条件分组
- 如果不写这个， 那整个结果集都是一个分区， 进行后续计算
order by：定义窗口内行的顺序， 一些偏移函数需要
rows：基于行数划分窗口
- rows between 2 preceding and current row
range 基于order by的值划分窗口
- range between unbounded prededing and current now
