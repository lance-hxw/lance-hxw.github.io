# 线程创建

```java
1. 继承thread
class MyThread extends Thread{
	@Override
	public void run(){
		...
	}
}
MyThread t = new MyThread();
t.start();

2. 实现runnable
class MyRun implements Runnable{
	@Override
	public void run(){
	..
	}
}
t = new Thread(new MyRun());
t.start()


3. lambda
   
t = new Thread(() -> {..});
t.start()

4. callable
   
Callable<Integer> callable = () -> {
	return 10;
}
FutureTask<Intger> futureTask = new FutureTask<>(callable);
t = new Thread(futureTask);
t.start();

5. 线程池
   
ExecutorService executor = Executors.newFiexedThreadPool(3);
executor.submit(() -> {});


```
