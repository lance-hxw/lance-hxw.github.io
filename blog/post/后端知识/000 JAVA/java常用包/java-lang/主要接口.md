java lang中是java最基础的设计契约

主要有 ：
**核心接口**

| 接口                                | 作用                       | 常见用法                        |
| --------------------------------- | ------------------------ | --------------------------- |
| `Iterable<T>`                     | 可迭代容器，支持 for-each        | `for (T item : collection)` |
| `Comparable<T>`                   | 定义自然排序顺序                 | `String`、`Integer` 的排序      |
| `Runnable`                        | 定义一个无参无返回值的任务            | `Thread`、`ExecutorService`  |
| `Cloneable`                       | 标记接口，允许 `Object.clone()` | 很少用，一般用拷贝构造器替代              |
| `Readable`                        | 定义字符源读取                  | `Reader` 实现它                |
| `Appendable`                      | 定义字符追加目标                 | `StringBuilder`、`Writer`    |
| `AutoCloseable`                   | 定义可自动关闭的资源               | `try-with-resources` 语法     |
| `Thread.UncaughtExceptionHandler` | 处理线程未捕获异常                | 设置线程异常兜底                    |
